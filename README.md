# 🩺 Dr. Moaz Mohamed Saad – Medical Portfolio Website

A modern, responsive, and professional personal portfolio website designed and developed for **Dr. Moaz Mohamed Saad**, an Intern Doctor and Menoufia University Graduate.

The portfolio showcases academic achievements, clinical training, USMLE accomplishments, professional skills, certifications, and contact information through a clean medical-inspired user interface.

---

## 🌟 Overview

This project was created to establish a strong professional online presence for Dr. Moaz and provide a centralized platform where colleagues, recruiters, residency programs, and healthcare organizations can learn more about his background, qualifications, and achievements.

The website follows modern design principles with a focus on:

* Professional medical branding
* Clean and elegant UI
* Responsive experience across all devices
* Fast loading performance
* Accessibility and usability
* Easy future customization
